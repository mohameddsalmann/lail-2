# perfume_nspired

A [Next.js](https://nextjs.org) perfume recommendation app for **nspired Beauty** — personalized fragrance quiz and product discovery.

## Features

- **Personalized quiz** — Find your scent by gender, notes, season, and intensity
- **Recommendation engine** — Matches perfumes to your preferences with scores and reasons
- **Product catalog** — Perfumes data with notes, prices, and links to nspired Beauty
- **Responsive UI** — Tailwind CSS, Framer Motion, clean layout

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Use **Find Your Scent** to run the quiz.

## Scripts

- `npm run dev` — Development server
- `npm run build` — Production build
- `npm run start` — Start production server
- `npm run lint` — ESLint

## Deploy

Deploy on [Vercel](https://vercel.com) or any Node.js host. See [Next.js deployment docs](https://nextjs.org/docs/app/building-your-application/deploying).
